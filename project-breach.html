Into The Breach

A board game coded in Python leveraging OOP and Tkinter.

ROLE

Game Developer

TIMELINE

September 2024 - December 2024

The Game Concept

Into The Breach is a turn-based tactical grid game inspired by the popular indie game of the same name. Players command units to battle against enemies, navigating a grid-based map and using strategic positioning to overcome challenges.

Development Process

This project was built using Python with Tkinter for the graphical user interface. I employed object-oriented programming principles to create a clean, modular codebase that separates game logic from the visual presentation.

Key components of the implementation include:

  • Grid-based movement system
  • Turn-based combat mechanics
  • Unit stats tracking (HP, damage, etc.)
  • Simple AI for enemy movement

Game Mechanics

Unit Types

The game features different unit types with unique abilities and stats:

  • Tanks: High HP, moderate damage
  • Artillery: Long range, area damage
  • Support: Healing and buffs

Combat System

Combat is resolved through a simple calculation based on unit stats and position. Attacking from behind or the sides provides bonuses, encouraging strategic positioning.

Technical Challenges

One of the biggest challenges was implementing the grid-based movement and combat system. I had to ensure that units could only move to valid positions and that combat calculations accurately reflected the game design.

Another challenge was creating a responsive UI with Tkinter, which required careful planning of the layout and event handling to ensure smooth gameplay.

What I Learned

This project deepened my understanding of:

  • Object-oriented programming in Python
  • Event-driven programming with Tkinter
  • Game design principles and balance
  • Implementing algorithms for grid-based movement and combat
