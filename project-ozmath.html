A platform designed to propel students ahead of their competition.

The inspiration

I've always enjoyed Mathematics ever since I was young. I also enjoy coding and tracking problems. So I thought I could combine the two! I took a lot of inspiration from Leetcode, which is another problem-based learning website catered for coding problems. I wanted to do the same however for mathematics. So I made OzMath!

The process

OzMath is a React project written primarily in TypeScript. The user database handling and authentication is managed by Supabase, the payment is handled by Stripe, and the website itself is hosted by Netlify. I use MDX Video to handle the media and Framer to design and host the landing page. I use the OpenAI API and Google OAuth API to grade responses and authenticate users. ShadowUI was a big help setting up the sidebar and I used their react-resizable implementation to allow users to resize the containers in the problem view.

Challenges & Outcomes

This project sort of blew up. I began it hoping to develop and demonstrate my skills in software development, however with every new addition, I became increasingly more passionate and saw potential for the project to become more so a start-up.

I'm still working on OzMath at the moment, mainly doing data entry converting problems to a JSON to add to the problems database.

Throughout my journey developing OzMath, I ran into a lot of challenges - the main ones were probably the resizable container and latex wrapping issues, which I have documented on my Blog.
